# README

This project is the Web application of _eigakan_ system for managing videos, movies, series, etc.

## Objectives

I created this project to organize all video media in a centralized storage and serve it in my homelab. It should work like a local _Youtube_ or _Netflix_.

## Getting Started

First install the dependencies:

```bash
npm install
```

To execute the project run the following command:

```bash
npm run dev
```
